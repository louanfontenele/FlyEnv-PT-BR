# FlyEnv UI 交互与功能详解指南

> 本指南按照你使用 FlyEnv 的自然顺序展开，帮助你发现界面中那些"看不见"的高效功能。

---

## 第一章：初见 FlyEnv —— 界面与交互语言

### 1.1 主界面布局

FlyEnv 采用经典的三栏布局：

- **左侧边栏**：模块导航、全局控制、设置入口
- **顶部标题栏**：当前模块的名称与快捷操作
- **主内容区**：当前选中模块的功能面板

<!-- [UI截图占位: 主界面全貌，标注左侧边栏、顶部标题栏、主内容区] -->

### 1.2 左侧边栏

操作路径：`主界面 -> 左侧边栏`

#### 折叠与展开
- 点击边栏顶部的折叠图标，可将导航栏收起为仅显示图标的窄栏，释放更多主内容区空间。
- 当窗口宽度小于 960 像素时，展开按钮会自动置灰不可用；若窗口从宽变窄到该阈值以下，边栏也会自动收起。
- 收起状态下，悬停模块图标即可看到模块名称。

<!-- [UI截图占位: 边栏展开与收起两种状态的对比] -->

#### 批量启停
- 边栏顶部提供一键启停图标，可控制所有可见服务的启动或停止。
- 若当前有服务正在获取状态，该按钮会自动置灰禁用，防止在状态未就绪时误操作。

#### 应用日志入口
- 点击日志图标打开应用日志抽屉。日志使用 Monaco 编辑器展示。
- **智能滚动**：若你正处于日志底部，新日志到达时会自动滚动到底部；若你手动向上翻阅了历史内容，新日志到达时不会打断你的阅读位置。

### 1.3 通用交互习惯（交互黑话表）

FlyEnv 有一套贯穿全软件的交互语言。掌握以下 6 个习惯后，可快速上手任何模块。

| 习惯 | 用户动作 | 典型出现位置 |
|---|---|---|
| **悬停显隐** | 鼠标悬停在某行或卡片上，原本隐藏的操作按钮才会浮现 | 服务版本表格、Node.js 脚本列表、自定义工具卡片 |
| **暗色 Popover** | 点击行末的 `...` 图标，弹出暗色无箭头的操作菜单 | 站点列表、服务版本表、容器列表 |
| **点击复制** | 悬停时文字变为黄色，点击即可复制到剪贴板 | 数据库密码、容器名、ApiToken、证书路径 |
| **双击快速编辑** | 双击表格行直接进入行内编辑，点击行外保存或取消 | 站点列表、语言项目列表、数据库备注 |
| **延迟提示** | 悬停在问号图标或按钮上片刻，显示说明气泡 | 设置项、工具栏按钮、状态图标 |
| **配置编辑器** | Raw File / Common Settings 双视图、保存红点、文件选择器 | 所有服务的 Config 入口 |

> 本章之后，文档中遇到上述模式**不再重复展开**，仅标注"参见 1.3 节交互习惯"。

## 第二章：设置

操作路径：`主界面 -> 左侧边栏底部 -> 点击设置图标`

设置中心分为多个标签页：Base、Module、UI、Licenses、About。本章聚焦于与日常操作密切相关的设置项。

### 2.1 基础设置（Base）

- 大量设置项右侧配有问号图标，悬停片刻会弹出气泡说明该选项的作用。

<!-- [UI截图占位: 基础设置页中某个设置项的悬停提示气泡] -->

### 2.2 模块管理（Module）

这是控制左侧边栏模块显隐的核心区域。

#### 分组级显隐开关
操作路径：`设置中心 -> Module 标签页 -> 点击分组标题旁的开关`

- 每个分组（如 Web Server、Database）都有一个总开关，可一键显示或隐藏该分组下的所有模块。

#### 模块级显隐开关
操作路径：`设置中心 -> Module 标签页 -> 点击单个模块卡片上的开关`

- 单独控制某一个模块是否在左侧边栏显示。
- **深层行为**：关闭自定义模块的显示开关时，如果该服务正在运行，系统会先停止该服务，再将其隐藏。

#### 悬停显隐的编辑入口
操作路径：`设置中心 -> Module 标签页 -> 悬停于自定义模块卡片上`

- 自定义模块卡片在默认状态下只显示名称和开关。
- 悬停后，卡片右侧会浮现**编辑**和**删除**按钮。

#### 自定义分类管理
操作路径：`设置中心 -> Module 标签页 -> 悬停于第一个分组标题上`

- 第一个分组（通常为 Site）标题旁有一个 **"+" 按钮**，用于添加自定义模块分类。
- 已创建的自定义分类在悬停时，标题右侧会显示**编辑**和**删除**图标。

<!-- [UI截图占位: Module 管理页，展示悬停浮现编辑/删除按钮的效果] -->

### 2.3 其他标签页

- **UI**：编辑器主题、字体等界面外观设置。
- **Licenses**：许可证绑定与管理。
- **About**：应用版本与作者信息。

## 第三章：服务与版本

### 3.1 版本管理通用模式

FlyEnv 的大部分标准服务模块（Nginx、Apache、PHP、MySQL、Redis、PostgreSQL、MongoDB 等）复用同一套 UI 架构。理解以下通用模式后，可快速上手任何新模块。

<!-- [UI截图占位: 某个标准服务模块（如 Nginx）的主界面全貌] -->

#### 模块主界面布局
标准服务模块的主界面通常分为三个区域：左侧 Aside（状态控制）、顶部 Tab（Service / Version Manager 切换）、主内容区（版本列表或配置面板）。

#### 版本列表表格
操作路径：`某服务模块 -> Service 标签页 -> 观察版本列表`

- **版本号点击打开目录**：表格中的版本号文本可直接点击，在系统文件管理器中打开该版本的安装目录。
- **当前版本高亮**：与左侧 Aside 中选中的当前版本相匹配的行，版本号和路径会显示为蓝色。
- **无效版本路径提示**：若某个版本的路径失效，路径文字会变为红色；悬停片刻会弹出错误说明气泡。表格前 3 行的气泡会自动从下方弹出，避免被顶部截断。
- **快速启动过滤**：仅在当前被选中的版本行显示一个特殊图标，点击可决定该版本是否参与边栏的"一键启停"。

#### 环境变量状态（Env 列）
- **蓝色图标**：由 FlyEnv 设置的环境变量
- **黄色图标**：系统中已存在该环境变量，但不是由 FlyEnv 设置的
- **灰色/隐藏**：未加入 PATH
- 当版本未加入 PATH 时，一个设置按钮默认隐藏，仅悬停在该行时才会浮现（参见 1.3 节交互习惯）。

#### 行末操作菜单（`...` Popover）
- 点击任意行末的 `...` 图标，弹出暗色无箭头 Popover，内含：
  - Open：打开版本所在目录
  - Add to Path：加入环境变量
  - Set Alias：设置别名
  - Config / Log / Extension 等：视具体模块而定
  - Delete：删除该版本（仅 Windows 显示）

#### 版本管理器（Version Manager）
操作路径：`某服务模块 -> 点击 Version Manager 标签页`

- **平台感知的数据源切换**：不同操作系统显示的选项不同：
  - macOS：Static / Homebrew / MacPorts / SDKMAN
  - Windows：仅 Static
  - Linux：Static / Homebrew / SDKMAN
- **自定义版本目录**：卡片标题右侧的文件夹加号图标，悬停片刻显示说明；点击后可添加自定义扫描目录，FlyEnv 会额外扫描这些目录下的已安装版本。

#### 配置编辑器
操作路径：`某服务模块 -> 点击 Config 按钮`

所有服务的配置编辑器采用统一界面（参见 1.3 节交互习惯）：
- Raw File / Common Settings 双视图切换
- 保存按钮在未修改时无标记，有变更后旁会出现红色小圆点
- 文件/目录输入框右侧配有文件夹图标，调用系统原生对话框
- Common Settings 视图下，每个设置卡片标题旁的问号图标悬停显示说明

<!-- [UI截图占位: 配置编辑器抽屉，展示保存红点、视图切换、文件选择器] -->

### 3.2 Web 服务器（Nginx / Apache / Caddy / FrankenPHP）

各 Web 服务器模块的交互差异极小，核心均遵循 3.1 节的通用模式。以下是其特有功能：

- **配置重载**：在 Vhost 编辑器中按 `Ctrl/Cmd + S` 保存配置后，会自动重载对应的 Web 服务（详见 4.6 节）。
- **站点日志动态 Tab**：日志类型根据已安装服务动态显示（详见 4.7 节）。

### 3.3 PHP 与扩展

#### 扩展管理器
操作路径：`PHP 模块 -> 点击 Extension 按钮`

- 以抽屉形式打开。
- **平台感知 Tab**：macOS 显示 Homebrew / MacPorts / Flyenv / Default；Windows 显示 Local / Lib。不同平台看到的入口不同。

### 3.4 Node.js 项目管理

操作路径：`Node.js 模块 -> 观察项目列表`

- **脚本播放按钮悬停显隐**：`npm` / `yarn` / `pnpm` 脚本列表中，每行的播放按钮默认不可见，仅悬停时浮现（参见 1.3 节交互习惯）。
- **依赖包点击打开 npm 页面**：点击依赖包名称，直接在浏览器中打开该包在 npmjs.com 的页面。
- **命令文本点击复制**：点击脚本命令，自动复制到剪贴板。
- **行末菜单的平台差异**：Windows 额外显示 PowerShell / PowerShell7+ 选项；macOS / Linux 显示 Terminal 选项。

<!-- [UI截图占位: Node.js 项目列表中悬停浮现播放按钮的状态] -->

### 3.5 数据库（MySQL / MariaDB / PostgreSQL）

#### 数据库管理抽屉
操作路径：`MySQL 或 MariaDB 模块 -> Service 标签页 -> 点击数据库管理按钮`

- 以抽屉形式打开。
- 若服务未运行，抽屉内会显示空状态提示，引导你先启动服务。

#### 顶部工具栏
| 按钮 | 说明 |
|---|---|
| Add Database | 新增数据库（弹出对话框） |
| Root Password | 修改 root 密码（弹出对话框） |
| phpMyAdmin | 打开内置的 phpMyAdmin 网页管理界面 |
| Database Backup | 备份选中的数据库；下拉可设置备份保存目录 |

<!-- [UI截图占位: MySQL 数据库管理抽屉的顶部工具栏] -->

#### 数据库列表交互
- **行内备注编辑**：悬停在某行的备注上，文字会变为输入框，直接输入后按回车或失焦即可自动保存（参见 1.3 节交互习惯）。
- **点击复制**：点击数据库名称、用户名或密码，自动复制到剪贴板（参见 1.3 节交互习惯）。
- **密码显隐切换**：密码列默认掩码显示，点击眼睛图标可切换显示/隐藏。

<!-- [UI截图占位: 数据库列表中点击复制、密码显示切换、行内备注编辑的对比状态] -->

### 3.6 容器与编排（Podman）

#### Machine 管理
操作路径：`Podman 模块 -> 左侧 Machine 列表`

> Linux 系统不显示 Machine 模块，因为 Linux 原生支持容器，无需虚拟机。

- Machine 列表行悬停时背景高亮。
- 每行右侧的 `...` 更多按钮默认隐藏，仅悬停时浮现（参见 1.3 节交互习惯）。
- 下拉菜单根据 Machine 的运行状态动态切换：运行时显示 Stop，停止时显示 Start。

#### Container / Image / Compose 通用交互
- **点击复制**：容器名、镜像名、Compose 路径等文本支持点击复制，悬停时文字变为黄色（参见 1.3 节交互习惯）。
- **状态联动**：每行的状态图标遵循优先级——错误提示 → 加载动画 → 停止图标（运行中）/ 播放图标（已停止）。
- **执行中按钮 reopen 终端**：当容器正在执行拉取/构建等操作时，操作列会显示 "executing" 按钮，点击可重新打开终端对话框查看实时输出。
- **Compose 动态菜单**：Compose 文件的操作下拉菜单会根据 `docker-compose.yml` 同级目录下的实际文件，**自动生成** "Edit `<文件名>`" 条目。

<!-- [UI截图占位: Podman Compose 列表的动态下拉菜单，展示自动生成的文件编辑入口] -->

#### 右侧详情面板
- Dashboard、Compose、Image、Container 四个标签页仅在 Machine 运行时才能查看；Machine 停止时显示空状态提示。

### 3.7 网络与隧道（Cloudflare Tunnel）

操作路径：`Cloudflare Tunnel 模块 -> 观察隧道列表`

- 每条隧道行默认**展开显示其下的 DNS 规则子表**。
- 子表中的子域名链接颜色随隧道运行状态变化：运行中为绿色，停止时为灰色。
- ApiToken 列和 TunnelName 列支持点击复制（参见 1.3 节交互习惯）。

#### 新增隧道
- **Zone 自动获取**：在 ApiToken 输入框中输入有效令牌后，系统会自动向 Cloudflare 拉取 Zone 列表。
- **级联填充**：选择 Zone 后，Zone Name 和 Account ID 会自动填充。
- **重复校验**：在线域名会进行全局唯一性校验，若已存在则标红提示。
- **本地主机自动补全**：Local Host 输入框支持从 FlyEnv 已有的站点和别名中自动补全。

<!-- [UI截图占位: Cloudflare Tunnel 新增对话框，展示 Zone 下拉和级联自动填充] -->

### 3.8 DNS 服务

操作路径：`DNS 模块 -> Service 标签页`

- 当 DNS 服务运行时，IP 选择下拉框的文字会强制显示为**绿色**，直观提示服务活跃。
- IP 旁配有问号图标，悬停片刻显示说明气泡。
- 若配置文件中的绑定 IP 不是 `0.0.0.0`，下拉框会被替换为静态只读标签，显示实际绑定的固定 IP。
- 日志区域使用虚拟表格渲染，固定列显示请求时间、类型、域名、IP 等信息。

<!-- [UI截图占位: DNS 服务运行状态下的绿色 IP 选择器和日志表格] -->

### 3.9 证书管理（MkCert）

#### 证书列表
操作路径：`MkCert 模块 -> Certificates 标签页`

- 证书路径和密钥路径支持点击打开所在文件夹，悬停时文字变为黄色（参见 1.3 节交互习惯）。
- 若站点没有显式配置 SSL 证书，列表会展示 FlyEnv 自动计算的默认证书路径。
- 顶部可选择使用哪个 mkcert 二进制版本来管理证书。

#### CA 根证书安装
操作路径：`MkCert 模块 -> 点击 Setup 或 Install CA`

- 界面依次展示三个状态：**Checking**（检测中）→ **Installed**（已安装）/ **Not Installed**（未安装）。
- CA 证书路径旁配有**复制**和**打开文件夹**按钮。
- 安装过程底部按钮随进度切换状态，安装界面内嵌 XTerm 终端，可查看 mkcert 命令的实时输出。

<!-- [UI截图占位: MkCert CA 安装界面，展示状态机和内嵌终端] -->

### 3.10 工具箱（Tools）

操作路径：`主界面 -> 点击右下角工具箱悬浮按钮 -> 打开工具箱抽屉`

工具箱以抽屉形式打开，左侧为树形导航，右侧为工具内容区。

#### 工具导航
- 点击顶部折叠按钮可收起左侧导航栏。
- 树形导航仅点击节点左侧的**展开箭头**才能折叠/展开分类；点击节点名称直接加载工具。
- 顶部搜索框支持跨所有工具实时过滤，选择建议项后直接跳转。

#### 工具卡片交互
- **收藏**：悬停在工具卡片上，左上角会浮现星标图标，点击即可收藏或取消收藏。收藏的工具会在首页 "Favorite Tools" 区域置顶显示。
- **自定义工具**：自定义工具卡片悬停时，右上角浮现更多下拉菜单（编辑 / 删除）。点击顶部 `+` 按钮可添加新自定义工具。

<!-- [UI截图占位: 工具箱首页展示收藏星标和自定义工具的下拉菜单] -->

#### Code Library
操作路径：`工具箱 -> 点击 Code Library`

- 点击顶部 **Setting** 齿轮图标，弹出气泡面板，可勾选控制各编程语言 Tab 的显示与隐藏。
- 悬停在分组标题或代码项上，右侧浮现 `...` 图标，点击显示 Edit / Move to Top / Delete / Move to Group。
- **批量操作模式**：从单个代码项的下拉菜单中选择 Batch Operations，进入多选模式，支持全选、批量移动到分组、批量删除。退出批量操作后，所有选中状态自动清空。
- 选中的代码项以 Markdown 形式渲染，代码块右上角配有复制按钮。

<!-- [UI截图占位: Code Library 的批量操作模式，展示复选框和批量操作按钮] -->

#### Code Playground
操作路径：`工具箱 -> 点击 Code Playground`

- **拖拽分栏**：左右两个编辑器之间有一条拖拽条，按住拖动可调整左右分栏比例，系统会自动记忆你的偏好。
- **可编辑 Tab**：支持新增和删除 Tab；第一个 Tab 受保护，不可关闭。
- **文件导入识别**：通过文件对话框导入代码文件时，系统会根据扩展名（`.py`、`.php`、`.go`、`.rs` 等）自动识别语言并加载内容。
- **输出格式转换**：右侧输出面板支持将运行结果转换为 Raw、JSON、YAML、XML、TOML、Go Struct、Rust Serde、Java、Kotlin、MySQL、JSDoc 等多种格式。
- **保存到 Code Library**：点击 Save 按钮，可将当前代码直接保存到 Code Library。

<!-- [UI截图占位: Code Playground 左右分栏拖拽调整宽度的状态] -->

#### 截图工具（Capturer）
- **条件快捷键捕获**：仅当鼠标悬停在截图工具卡片上时，才会监听键盘快捷键；鼠标移出后自动停止监听，避免全局冲突。
- 支持捕获组合键（Meta / Shift / Control / Alt + 任意键）。
- 两种截图模式：直接截图 / 隐藏 FlyEnv 窗口后截图（避免窗口遮挡目标区域）。
- 文件名规则支持插入变量：`{index}`、`{timestamp}`、`{datetime}`、`{uuid}`。

#### 图片压缩（Image Compress）
- 支持将图片或文件夹拖拽到批量处理卡片区域，整个卡片会变为虚线高亮边框。
- **平台差异化按钮**：macOS 仅显示一个 `+` 按钮；Windows / Linux 显示独立的 Picture 和 Folder 按钮。
- 批量多选删除（表格复选框）。
- 预览模式下，通过**竖向分割线**左右滑动，可直观对比压缩前后的画质。支持鼠标拖拽和触摸屏拖拽。

### 3.11 AI 助手

操作路径：`主界面 -> 点击右下角 AI 悬浮按钮 -> 打开 AI 聊天面板`

AI 面板以抽屉形式打开，分为 **FlyEnv**（任务型）和 **Ollama**（对话型）两种模式。

#### 聊天列表
- 悬停在历史会话列表项上，右侧浮现渐变遮罩和三点菜单（编辑标题 / 复制会话 / 删除）。
- 当聊天内容向上滚动超过一定距离时，右下角自动浮现**回到底部**按钮；点击可快速滚动至最新消息。
- 若你正处于底部附近，收到新消息时会自动下滚；若你手动向上翻阅历史，则不会被打断。

<!-- [UI截图占位: AI 聊天面板中悬停会话项浮现三点菜单，以及回到底部按钮] -->

#### 消息交互
- **助手消息**：悬停浮现 TTS 播放按钮和复制按钮。
- **用户消息**：悬停浮现复制按钮和**重新编辑**按钮；点击重新编辑会将该消息内容注入回输入框。
- **生成中停止**：AI 正在生成回复时，发送按钮会变为停止按钮，点击可立即中断。

#### 输入框快捷键
| 按键 | 作用 |

| Ctrl + Enter | 发送（不触发生成） |

#### 附件
- 点击输入框旁的**文档按钮**，可选择本地文件。超过 2MB 的文件会被自动过滤，文档内容以代码块形式插入消息。
- 点击**图片按钮**可附加图片进行多模态对话。

#### Prompt 管理器
操作路径：`AI 面板 -> 点击输入框旁的 Prompt 按钮`

- 点击触发（非悬停），弹出气泡面板，分 **Custom**（自定义）和 **FlyEnv**（预设）两栏。
- 悬停在任意 Prompt 单元格上，右上角浮现更多菜单（编辑 / 复制 / 删除 / 另存为自定义）。
- 顶部语言筛选器可按语言过滤 FlyEnv 预设 Prompt 列表。

#### FlyEnv 任务模式
- AI 直接在聊天流中嵌入交互组件（如文件夹选择器、站点选择器、故障排查按钮），无需跳出面板即可完成操作。
- 当任务需要等待用户输入时，输入框会暂停发送，直到你提供所需信息。

## 第四章：站点管理（Host）

站点管理是 FlyEnv 交互密度最高的区域，也是大多数用户的日常核心工作流。

### 4.1 顶部工具栏

操作路径：`主界面 -> 左侧边栏 -> 点击 Host 模块 -> 观察顶部工具栏`

#### 项目类型切换
操作路径：`顶部工具栏 -> 点击当前项目类型下拉按钮 -> 选择目标类型`

- 下拉菜单按分组展示项目类型，分组标题为灰色不可点击的说明文字，下方的子项（PHP、Tomcat 等）才可点击切换。
- 切换后，下方列表会展示对应类型的站点。

#### 添加站点与更多操作
- **Add 按钮**：打开新增站点抽屉。
- **"..." 更多按钮**：点击后展开下拉菜单，包含：
  - Export：导出所有站点配置（无站点时禁用）
  - Import：导入站点配置文件
  - Hosts Copy：复制当前 hosts 内容到剪贴板
  - Hosts Open：在系统文件中打开 hosts 文件
  - Vhost Template：打开虚拟主机模板管理（详见 4.8 节）
  - New Project：新建项目向导
    - 悬停在 "New Project" 上片刻，会弹出气泡说明该功能的作用。

#### Hosts 文件写入控制
- **Enable 开关**：控制是否将 FlyEnv 管理的站点自动写入系统 hosts 文件。悬停片刻显示说明气泡。
- **IPv6 开关**：仅当 Enable 开启后才会出现。控制写入 hosts 时是否同时生成 IPv6 映射条目。

#### 打开系统 Hosts 编辑器
- 点击 OpenHosts 按钮，直接打开系统 hosts 文件的 Monaco 编辑器。
- **快捷键**：在编辑器内按 `Ctrl/Cmd + S` 可直接保存系统 hosts 文件。

### 4.2 站点列表

#### 自动层级展开
- 站点列表并非纯平铺，而是根据域名后缀自动计算父子关系并层级展开。
- 例如：若同时拥有 `example.test` 和 `sub.example.test`，后者会自动嵌套在前者下方。
- 所有层级默认处于展开状态，可点击行首箭头手动折叠。

<!-- [UI截图占位: 站点列表展示层级嵌套关系] -->

#### 双击快速编辑
操作路径：`站点列表 -> 双击任意一行`

- 双击表格任意一行，立即进入该行的**行内快速编辑模式**（参见 1.3 节交互习惯）。
- 可快速修改：站点名、PHP 版本、备注。
- 点击行外任意区域退出编辑；若内容有变化，自动保存；若无变化，直接取消。
- 若输入的站点名与现有站点重复，输入框会标红提示。
- 空备注单元格悬停时，会浮现 faint yellow 提示文字 "Double click row to edit"，引导用户发现该功能。

<!-- [UI截图占位: 双击某行后进入行内编辑模式的对比图] -->

#### QR Code 调试
操作路径：`站点列表 -> 悬停于站点名上约 1 秒`

- 悬停在站点名称上片刻，会弹出气泡，内含该站点 URL 的 **QR Code**。
- 气泡右下角有下载图标，点击可将 QR Code 保存为 PNG，方便手机扫码调试。
- 站点名左侧的链接图标只有在 "Hosts 写入" 开启且至少有一个 Web 服务器（Nginx/Apache/Caddy）在运行时，才会变为高亮可点击状态；否则为灰色。

<!-- [UI截图占位: 悬停站点名弹出 QR Code 气泡的完整状态] -->

#### 长备注查看
- 当备注文字过长被截断时，悬停片刻会弹出 Tooltip，展示完整内容。

#### 行末操作菜单
操作路径：`站点列表 -> 点击任意行末的 "..." 图标`

- 点击后展开暗色无箭头 Popover（参见 1.3 节交互习惯），内含：

| 操作项 | 说明 |
|---|---|
| Open | 在系统文件管理器中打开站点根目录 |
| Edit | 打开站点编辑抽屉 |
| Park | 启用 Park 模式（子目录站点自动识别） |
| Links | 查看该站点的所有可用链接 |
| Config File - Nginx | 仅当 Nginx 已安装时显示 |
| Config File - Caddy | 仅当 Caddy 已安装时显示 |
| Config File - Apache | 仅当 Apache 已安装时显示 |
| Config File - FrankenPHP | 仅当 FrankenPHP 已安装时显示 |

> 操作菜单中的 "Config File" 项会根据你已安装并启用的 Web 服务器**动态显示或隐藏**（参见 1.3 节交互习惯）。

### 4.3 排序与置顶（Sort）

操作路径：`站点列表 -> 点击某行 "..." -> 点击 Sort`

- 点击 Sort 后，会在该行附近浮现一个悬浮面板。
- 面板内包含两个控制：
  - **Sticky 开关**：开启后，该站点移动到列表最顶部的"置顶区"；关闭后回到普通列表区。
  - **纵向滑块**：在各自的区域（置顶区或普通区）内，拖动滑块调整站点的前后顺序。
- 关闭面板（点击面板外区域）时，排序结果自动保存，并平滑滚动到被排序的站点位置。

<!-- [UI截图占位: Sort 弹出面板，展示 Sticky 开关和纵向滑块] -->

### 4.4 新建/编辑站点

操作路径：`顶部工具栏 -> 点击 Add / 或点击某行 Edit`

#### 基础信息
- **Site Name**：站点域名，必填。
- **Alias**：站点别名，支持多行，每行一个别名域名。
- **Comment**：备注，用于在列表中标注站点用途。
- **Root Path**：站点根目录，输入框右侧的文件夹图标可调用系统原生目录选择对话框。

#### Park 模式
- 开关右侧配有问号图标，悬停显示 Park 功能的详细说明。
- **开启确认**：启用 Park 时会弹出二次确认对话框，防止误操作。

#### PHP 版本
- 下拉选择该站点使用的 PHP 版本。
- **运行状态可视化**：下拉列表中，当前正在运行的 PHP 版本会显示为**绿色**，方便直接选择活跃环境。
- **静态站点**：选择第一项可将该站点作为纯静态站点运行，不经过 PHP 解析。

<!-- [UI截图占位: 编辑站点抽屉中的 PHP 版本下拉框，展示绿色高亮的运行中版本] -->

#### 端口配置
- 分别设置 Nginx、Caddy、Apache 的 HTTP 端口（默认 80）。
- 若输入非法或冲突，输入框会标红。

#### SSL / HTTPS
- **SSL 总开关**：开启后才显示下方的证书配置。
- **Auto SSL**：开启后，FlyEnv 自动使用内置 mkcert 根证书为该站点签发本地信任证书，无需手动填写证书路径。
- **手动证书**：关闭 Auto SSL 后，需手动填写 Cert 和 Cert Key 路径。输入框右侧的文件夹图标可调用系统文件选择对话框（此处选择的是文件，而非目录）。
- **SSL Port**：开启 SSL 后，显示 Nginx/Caddy/Apache 的 HTTPS 端口配置（默认 443）。

<!-- [UI截图占位: SSL 配置区域，展示 AutoSSL 开关和手动证书路径选择] -->

#### SSL 提示气泡
- Auto SSL 标签旁的问号图标悬停后，会弹出大型气泡，内含：
  - 信任根 CA 的命令行说明
  - Copy 按钮：复制命令到剪贴板
  - Show CA File 按钮：在文件夹中定位 CA 证书文件

<!-- [UI截图占位: 悬停 SSL 问号图标后弹出的大型提示气泡] -->

#### Nginx Rewrite 规则
- 支持直接输入 Nginx 重写规则。
- 输入框右侧有一个下拉按钮，可将当前重写规则文件直接在 **VSCode、PhpStorm、WebStorm 或 Sublime Text** 中打开编辑。

#### 反向代理
- 点击标题右侧的 `+` 按钮，可添加反向代理规则。
- 每条规则包含：路径匹配（Path）和目标地址（URL）。
- 点击规则左侧的删除图标可移除该条代理。

#### 保存前的校验
- 若创建新站点时未选择 PHP 版本，点击 Save 会弹出确认对话框，提醒你确认是否以静态站点方式创建。
- 必填字段（如站点名、根目录）若为空，对应输入框会标红，阻止保存。

### 4.5 虚拟主机配置编辑器（Vhost）

操作路径：`站点列表 -> 点击某行 "..." -> 点击 Config File - Nginx/Caddy/Apache/FrankenPHP`

- 以抽屉形式打开 Monaco 代码编辑器。
- **自动初始化**：若该站点的配置文件尚未生成，编辑器会自动触发配置初始化，生成默认配置文件后再加载内容。
- **快捷键保存**：在编辑器内按 `Ctrl/Cmd + S`，会直接保存配置文件内容到磁盘，并**自动重载对应的 Web 服务器**，无需手动重启服务。
- 工具栏提供两个按钮：Open（在文件管理器中定位配置文件）、Save（手动保存）。

<!-- [UI截图占位: Vhost 配置编辑器抽屉，展示 Monaco 编辑器和工具栏] -->

### 4.6 站点日志

操作路径：`站点列表 -> 点击某行 "..." -> 点击 Log`

- 以抽屉形式打开，展示该站点相关的服务器日志。
- **动态日志类型**：顶部 Tab 栏根据你已安装并运行的 Web 服务器动态显示。例如仅安装了 Nginx 则只显示 Nginx Log；同时安装了 Nginx 和 Caddy，则显示多个日志类型。
- **偏好记忆**：你上次选中的日志类型会被记住，下次打开时自动恢复。

<!-- [UI截图占位: 站点日志抽屉，展示动态 Tab 和日志内容] -->

### 4.7 站点链接查看器（Links）

操作路径：`站点列表 -> 点击某行 "..." -> 点击 Links`

- 弹出一个对话框，自动汇总该站点的所有可用访问地址：
  - HTTP 与 HTTPS 链接
  - 所有别名对应的链接
  - 端口变体（自动省略 :80 和 :443 的标准端口）
  - 对于 Node/Go/Python/Tomcat 类型的项目，直接使用项目的 projectPort 生成链接

### 4.8 虚拟主机模板管理（Vhost Template）

操作路径：`顶部工具栏 -> 点击 "..." 更多按钮 -> 点击 Vhost Template`

- 点击后打开悬浮面板，同时支持悬停触发和点击触发。
- 面板内提供以下模板编辑入口：Apache / Apache SSL / Nginx / Nginx SSL / Caddy / Caddy SSL。
- 选择任意一项，即可在 Monaco 编辑器中修改该类站点的默认虚拟主机模板，影响后续新增站点的初始配置。
